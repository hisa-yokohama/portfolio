# DESIGN.md — 横浜中華街 グルメガイド HP

## 1. コンセプト
横浜中華街の「今、流行っているお店」を紹介するグルメガイドサイト。
食欲をそそる写真的表現と、洗練された editorial（雑誌的）レイアウトを両立する。
旅行前・デート前に「どこで食べる？」を解決するキュレーションメディア。

- トーン: 上質・清潔・少しモダンチャイナ
- ターゲット: 20〜40代、週末に横浜へ出かける人、食べ歩き好き
- 想定デバイス: モバイルファースト（横浜散策中にスマホで見る）

## 2. デザイン原則（Claude Cookbook 美学プロンプト適用）
> "Be bold and intentional. Establish a clear visual hierarchy. Use whitespace
> generously. Choose a restrained palette and let one accent do the work.
> Typography is the interface — make it large, legible, and confident."

これを本サイトに翻訳:
1. **白ベース** — 背景は純白〜オフホワイト(#fafafa)。余白を主役にする。
2. **文字大きめ** — 本文 18px / 見出しは 48〜88px のディスプレイサイズ。
3. **アクセントは1色（中華朱）** — 朱赤 `#d6442f` ＋ 金 `#c9a24b` を差し色に限定。
4. **階層を明確に** — 大見出し→リード→本文の3段階。視線を迷わせない。
5. **動きは控えめに** — スクロールでふわっと出るフェードのみ。過剰演出はしない。

## 3. カラーパレット
| 用途 | 色 | HEX |
|------|----|-----|
| 背景 | オフホワイト | `#faf9f7` |
| 文字（主） | 墨 | `#1a1815` |
| 文字（副） | グレー | `#6b6660` |
| アクセント1 | 中華朱 | `#d6442f` |
| アクセント2 | 金 | `#c9a24b` |
| 罫線 | うすグレー | `#e7e3dd` |

## 4. タイポグラフィ
- 和文: "Noto Serif JP"（見出し・明朝で品格）＋ "Noto Sans JP"（本文）
- 欧文: "Playfair Display"（数字・英字見出し）
- スケール: H1 88px / H2 48px / リード 22px / 本文 18px / キャプション 14px
- 行間: 本文 1.9、見出し 1.2

## 5. レイアウト構成（セクション単位＝実行単位）
1. **Header** — ロゴ「横濱中華街」＋ナビ。透明→白追従。
2. **Hero** — 大きな見出し「いま、食べたい中華街。」＋リード＋朱の縦帯装飾。
3. **Intro** — 中華街の魅力を3つの数字で（店舗数/年間来訪者/エリア）。
4. **Featured** — 流行りのお店カード（4〜6軒）。写真枠＋ジャンルタグ＋一言。
5. **Categories** — 食べ歩き/点心/火鍋/スイーツ などジャンル別グリッド。
6. **Access** — アクセス情報・最寄駅・散策のコツ。
7. **Footer** — クレジット・注記。

## 6. インタラクション
- スクロールイン時に `opacity/translateY` のフェードアップ（IntersectionObserver）。
- カードホバーで微拡大＋影。
- ヘッダーはスクロールで白背景＋影を付与。

## 7. 注記
掲載店舗・データはデザイン提示用のサンプル（representative content）。
実掲載時は各店の許諾・最新の営業情報・写真権利を確認すること。
